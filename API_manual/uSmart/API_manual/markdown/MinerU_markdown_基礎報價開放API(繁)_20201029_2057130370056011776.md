# ⼀、統⼀HTTP開端

# 簡要描述

基礎報價的接口，每次調⽤都必須攜帶⼀些必要的HTTP開端，這些HTTP開端及其功能如下所示：

<table><tr><td>參數名</td><td>必選</td><td>類型</td><td>說明</td></tr><tr><td>Content-Type</td><td>是</td><td>string</td><td>必須包含:application/json</td></tr><tr><td>Authorization</td><td>是</td><td>string</td><td>登錄成功後返回的token</td></tr><tr><td>X-Channel</td><td>是</td><td>string</td><td>管道標記</td></tr><tr><td>X-Lang</td><td>是</td><td>int</td><td>語言類型:默認為簡體,1:簡體,2:繁體,3:英文</td></tr><tr><td>X-Request- Id</td><td>是</td><td>string</td><td>長度為19位元數位,必須確保唯一用於做幂等防重,推薦使用分散式 Snowflake雪花演算法生成</td></tr><tr><td>X-Time</td><td>是</td><td>string</td><td>unix 時間標記</td></tr><tr><td>X-Sign</td><td>是</td><td>string</td><td>簽名</td></tr></table>

# 簽名說明

X-Sign⽣成規則：

1. Authorization、 X-Channel、X-Lang、X-Request-Id、X-Time頭欄位與body內容按序拼接成原始內容rowContent.

2. 使⽤MD5withRSA演算法對rowContent進⾏摘要、加密⽣成密⽂cipherContent

3. 再對密⽂cipherContent使⽤safeBase64編碼⽣成X-Sign

# 六、逐筆接口

# 簡要描述

獲取逐筆資料的接口。

# 請求URL

https://open-hz.yxzq.com/quotes-openservice/api/v1/tick 

# 請求⽅式

POST 

# 請求參數

<table><tr><td>參數名</td><td>必選</td><td>類型</td><td>說明</td></tr><tr><td>secuId</td><td>是</td><td>string</td><td>證券的唯一標記,由市場標記+證券代碼構成,如騰訊的唯一標記為hk00700</td></tr><tr><td>tradeTime</td><td>是</td><td>int64</td><td>起始的報價時間,首頁可傳0,其他分頁傳遞結果中的最後或者最開始的一條的latestTime</td></tr><tr><td>seq</td><td>是</td><td>int64</td><td>起始的報價序號,首頁可傳0,其他分頁傳遞結果中的最後或者最開始的一條的req</td></tr><tr><td>count</td><td>int64</td><td>string</td><td>每頁數據的大小</td></tr><tr><td>sortDirection</td><td>int32</td><td>string</td><td>排序方向,0:時間逆序,1:時間順序</td></tr></table>


返回結果


<table><tr><td>參數名</td><td>類型</td><td>說明</td></tr><tr><td>code</td><td>int</td><td>錯誤碼,0表示成功,其他表示失敗</td></tr><tr><td>msg</td><td>string</td><td>消息</td></tr><tr><td>data</td><td>字典類型</td><td>資料區,包含一個逐筆條目的陣列,具體參照下面的逐筆條目說明</td></tr></table>


逐筆條目說明


<table><tr><td>參數名</td><td>類型</td><td>說明</td></tr><tr><td>seq</td><td>int32</td><td>報價序號</td></tr><tr><td>time</td><td>int64</td><td>報價時間</td></tr><tr><td>price</td><td>double</td><td>價格</td></tr><tr><td>volume</td><td>int64</td><td>成交量</td></tr><tr><td>direction</td><td>double</td><td>買賣方向, 0: 默認, 1: 買, 2: 賣</td></tr><tr><td>trdType</td><td>int64</td><td>逐筆類型, 港交所特有, 數值與類型之間的對應關係為: 4:P 22:M 100:Y 101:X 102:D 103:U</td></tr></table>

示例

```txt
POST /quotes-opensource/api/v1/tick HTTP/1.1
Content-Type: application/json; charset=utf-8
{"secuId":"sh600001","tradeTime":0,"start":0,"count":0,"sortDirection":0} 
```

```json
{
    "code": 0,
    "msg": "success",
    "data": {
    "market": "hk",
    "symbol": "00001",
    "start": "0",
    "list": [
    {
    "seq": 1202,
    "time": 20191224120820000,
    "price": 74.65,
    "volume": 1000,
    "direction": 0,
    "trdType": 103
    }
    ]
    }
} 
```

# 七、買賣盤接口

# 簡要描述

獲取買賣盤資料的接口。

# 請求URL

https://open-hz.yxzq.com/quotes-openservice/api/v1/orderbook 

# 請求⽅式

POST 

# 請求參數

<table><tr><td>參數名</td><td>必選</td><td>類型</td><td>說明</td></tr><tr><td>secuId</td><td>是</td><td>string</td><td>證券的唯一標記，由市場標記+證券代碼構成，如騰訊的唯一標記為hk00700</td></tr></table>


返回結果


<table><tr><td>參數名</td><td>類型</td><td>說明</td></tr><tr><td>code</td><td>int</td><td>錯誤碼,0表示成功,其他表示失敗</td></tr><tr><td>msg</td><td>string</td><td>消息</td></tr><tr><td>data</td><td>字典類型</td><td>資料區,包含一個最新報價時間(latestTime),一個買賣盤條目的陣列,具體參照下面的買賣盤條目說明</td></tr></table>


買賣盤條目說明


<table><tr><td>參數名</td><td>類型</td><td>說明</td></tr><tr><td>bidPrice</td><td>double</td><td>買盤價</td></tr><tr><td>bidVolume</td><td>int64</td><td>買盤量</td></tr><tr><td>bidOrderCount</td><td>int64</td><td>買委託訂單個數</td></tr><tr><td>askPrice</td><td>double</td><td>賣盤價</td></tr><tr><td>askVolume</td><td>int64</td><td>賣盤量</td></tr><tr><td>askOrderCount</td><td>int64</td><td>賣委託訂單個數</td></tr></table>

# 示例

```jsonl
POST /quotes-opensource/api/v1/orderbook HTTP/1.1
Content-Type: application/json; charset=utf-8
{"secuId":"hk00001"} 
```

```json
{
    "code": 0,
    "msg": "success",
    "data": {
    "market": "hk",
    "symbol": "00001",
    "latestTime": 20200109153121000,
    "list": [
    {
    "bidPrice": 9.31,
    "bidVolume": 12000,
    "bidOrderCount": 5,
    "askPrice": 9.32,
    "askVolume": 21600,
    "askOrderCount": 3
    }
    ]
    }
} 
```
