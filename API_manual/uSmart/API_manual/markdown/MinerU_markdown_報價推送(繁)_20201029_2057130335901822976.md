# 概述

盈立證券開放平台通過websocket與用戶端保持連接，從而提供即時報價推送服務，本文檔為

websocket連接接入指引，包括以下內容：

接入介面

常見錯誤碼定義

接口限制

市場標記定義

行情類型定義

證券狀態定義

# 介面

接入url: wss://open-hz.yxzq.com/wss/v1

接入協議: websocket

資料格式: json

# 心跳


伺服器ping


```json
{
    "op": "ping",
    "ts": 1575531634, //unix 時間戳
    "reqId": 666666 //unix 請求序號
}
```


用戶端pong


```json
{
    "op": "pong",
    "ts": 1575531634, //unix 時間戳
    "reqId": 666666 //unix 請求序號
}
```

# 授權


用戶端授權請求


```json
{
    "op": "auth",
    "ts": 1575531634, //unix 時間戳
    "reqId": 666666, //unix 請求序號，
    "accessToken": "4F65x5A2bLyMWVQj3Aqp+B4w+ivaA7n5Oi2SuYtCJ9o=",
}
```


伺服器授權應答


```json
{
    "op": "auth",
    "ts": 1575531634, //unix 時間戳"reqId":
    666666, //unix 請求序號"code": 0, //狀態碼成功
    "msg": "success" //狀態描述
}
```

# 訂閱

用戶端訂閱請求

```json
{
    "op": "sub",
    "ts": 1575531634, //unix 時間戳
    "reqId": 666666, //unix 請求序號
    "topiclist": ["$type.$market.$code",...] // 示例: ["rt.hk.00700", "tk.hk.00700", "ob.hk.00700"]
}
```


服務端訂閱應答


```json
{
    "op": "sub",
    "ts": 1575531634, //unix 時間戳"reqId":
    666666, //unix 請求序號"code": 0, //狀態碼成功
    "msg": "success" //狀態描述
}
```


用戶端取消訂閱請求


```json
{
    "op": "unsub",
    "ts": 1575531634, //unix 時間戳
    "reqId": 666666, //unix 請求序號
    "topiclist": ["$type.$market.$code",...] // 示例:["rt.hk.00700", "tk.hk.00700", "ob.hk.00700"]
}
```


服務端取消訂閱應答


```json
{
    "op": "unsub",
    "ts": 1575531634, //unix 時間戳"reqId":
    666666, //unix 請求序號"code": 0, //狀態碼成功
    "msg": "success" //狀態描述
}
```

# 報價推送


服務端主動推送報價


```json
{
    "op": "update",
    "topic": "$type.$market.$code", //内容所屬topic "data": {} //更新內容、根據即時，tick等具體業務定
}
```

<table><tr><td>參數名</td><td>類型</td><td>說明</td></tr><tr><td>seq</td><td>int32</td><td>報價序號</td></tr><tr><td>time</td><td>int64</td><td>報價時間</td></tr><tr><td>price</td><td>double</td><td>價格</td></tr><tr><td>volume</td><td>double</td><td>成交量</td></tr><tr><td>direction</td><td>double</td><td>買賣方向，0: 默認，1: 買，2: 賣</td></tr><tr><td>trdType</td><td>int64</td><td>逐筆類型，港交所特有，數值與類型之間的對應關係為：4:P 22:M 100:Y 101:X 102:D 103:U</td></tr></table>

    "direction": 2,
