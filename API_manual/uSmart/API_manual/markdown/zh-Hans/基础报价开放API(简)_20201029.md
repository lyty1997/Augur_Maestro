简要描述

基础报价的接口，每次调用都必須攜帶一些必要的HTTP开端，这些HTTP开端及其功能如下所示：

参数名 必
选

类型

说明
Content-
Type

是

string

必須包含：application/json
Authorization 是 string 登录成功后返回的token
X-Channel 是 string 渠道标記
X-Lang 是 int 語⾔类型：默认为简體，1：简體，2：繁體，3：英文
X-Request- Id

是

string 长度为19位元数位，必須确保唯一用于做冪等防重，推薦使用分散式
Snowflake雪花算法生成
X-Time 是 string unix 时间标記
X-Sign 是 string 签名

签名说明

X- Sign生成规则：
1. Authorization、 X-Channel、X-Lang、X-Request-Id、X-Time頭字段与body内容按序拼接成原始内容
rowContent.
2. 使用MD5withRSA算法对rowContent进行摘要、加密生成密文cipherContent
3. 再对密文cipherContent使用safeBase64编码生成X-Sign
